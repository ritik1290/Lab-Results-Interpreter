LabResult Interpretation

A simple, fast, and responsive web app that helps users understand basic laboratory test results using pre-defined interpretations. This project focuses on clean UI, smooth animations, and a visually engaging experience featuring a small monkey mascot. It is designed purely for educational and informational purposes.

Project Structure
/Accessories
    /Assets
        monkey.svg        → Small mascot icon used in UI
    /CSS
        style.css         → Styles, layout, and animations
    /JS
        script.js         → Core logic and UI interactions
data.json                 → Predefined lab ranges and basic interpretations
index.html                → Main application interface
